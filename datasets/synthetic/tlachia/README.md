# Tlachia Synthetic Fixtures

Fixtures sinteticos para demostrar ingesta multiplataforma sin API keys reales, llamadas externas ni datos personales.

Reglas:

- Todos los archivos deben declarar `synthetic: true`.
- Todos los archivos deben declarar `contains_real_personal_data: false`.
- Usar URLs `https://example.invalid/...`.
- No incluir nombres, handles, telefonos, correos o cuentas reales.
- Los escenarios son demostrativos y requieren revision humana.
